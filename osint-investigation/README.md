# Investigação OSINT

## Objetivo
Investigar informações públicas disponíveis na internet.

## Ferramentas utilizadas
- Google
- Maltego
- Spiderfoot

## Metodologia
Coleta e análise de informações públicas.

## Resultado
Foram identificadas informações públicas relacionadas ao perfil analisado.

## Conclusão
A investigação demonstrou como dados públicos podem revelar
informações relevantes.
