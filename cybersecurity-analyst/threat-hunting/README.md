# 🕵️‍♂️ Threat Hunting Portfolio

Bem-vindo ao meu portfólio de Threat Hunting, onde apresento investigações práticas voltadas à identificação de ameaças avançadas, análise comportamental e detecção proativa em ambientes corporativos.

**🎯 Objetivo**

Este repositório tem como objetivo demonstrar minhas habilidades em:

- Investigação proativa de ameaças (Threat Hunting)
- Análise de logs e eventos de segurança
- Detecção de atividades suspeitas e comportamentos anômalos
- Uso de ferramentas SIEM e frameworks de segurança
- Criação de hipóteses e validação baseada em dados
  
**🧠 Metodologia de Threat Hunting**

Os projetos seguem uma abordagem estruturada baseada em boas práticas da indústria:

Definição de hipótese<br>
Ex: "Usuários podem estar executando comandos suspeitos via PowerShell"<br>
Coleta de dados<br>
Logs de autenticação<br>
Eventos de endpoint<br>
Tráfego de rede<br>
Análise e investigação<br>
Correlação de eventos<br>
Identificação de padrões anômalos<br>
Query em SIEM (KQL, SPL, etc.)<br>
Validação<br>
Confirmação da ameaça ou falso positivo<br>
Documentação<br>
Registro detalhado do processo investigativo<br>

**🛠️ Ferramentas e Tecnologias**

- Microsoft Sentinel
- Splunk
- Wazuh
- Kusto Query Language (KQL)
- SPL (Search Processing Language)
- MITRE ATT&CK Framework
- Logs de segurança (Windows, Linux, Cloud)
