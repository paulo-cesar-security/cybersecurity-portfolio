# Log Analysis Portfolio

Este repositório reúne meus projetos e estudos práticos em Log Analysis (Análise de Logs) voltados à cibersegurança, com foco em investigação de eventos, detecção de anomalias e suporte à resposta a incidentes.

**🎯 Objetivo**

Demonstrar habilidades em:

Análise e interpretação de logs de segurança<br>
Identificação de padrões suspeitos e anomalias<br>
Correlação de eventos entre diferentes fontes de dados<br>
Suporte a investigações de incidentes (SOC / Blue Team)<br>
Uso de SIEM para consultas e detecção<br>
Criação de queries para hunting e investigação

**🧭 Escopo da Análise de Logs**

Os projetos incluem análise de logs provenientes de diferentes fontes:

🖥️ Logs de endpoints (Windows Event Logs, Sysmon)<br>
🌐 Logs de rede (firewalls, proxies, IDS/IPS)<br>
🔐 Logs de autenticação (Entra ID / Active Directory)<br>
☁️ Logs de cloud (Microsoft Defender, Azure logs)<br>
📧 Logs de e-mail (phishing analysis)
