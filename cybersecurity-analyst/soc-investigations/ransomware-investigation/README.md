# Laboratório SOC Analyst N1 — Investigação de Ransomware em Estágio Inicial

**Cenário**

Sou analista SOC Nível 1 da empresa fictícia LogiTrans.<br>
Durante meu turno, o SIEM gerou múltiplos alertas relacionados a:<br>
renomeação massiva de arquivos<br>
exclusão de shadow copies<br>
alto volume de escrita em disco<br>
O comportamento foi identificado em uma máquina do setor administrativo.<br>

**Meu objetivo era:**

validar possível ransomware<br>
identificar estágio do ataque<br>
verificar impacto inicial<br>
coletar IOC<br>
escalar rapidamente o incidente
