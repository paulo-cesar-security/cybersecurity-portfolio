# 🔐 Identity Security (Microsoft Entra ID) Portfolio

Este repositório reúne meus projetos e estudos práticos em Identity Security, com foco no Microsoft Entra ID (Azure Active Directory), abordando autenticação, autorização, controle de acesso e proteção de identidades em ambientes corporativos.

**🎯 Objetivo**

Demonstrar habilidades em:

Gestão de identidades e acessos (IAM)<br>
Configuração e administração do Microsoft Entra ID<br>
Implementação de políticas de segurança de identidade<br>
Proteção contra ataques baseados em credenciais<br>
Aplicação de princípios de Zero Trust<br>
Monitoramento e investigação de eventos de autenticação

**🧭 Áreas de Atuação** <br>
🆔 Identity & Access Management (IAM)<br>
Criação e gerenciamento de usuários e grupos<br>
Controle de permissões e funções (RBAC)<br>
Gestão do ciclo de vida de identidades

**🔐 Authentication & Authorization** <br>
Métodos de autenticação (MFA, passwordless) <br>
Políticas de acesso condicional<br>
Controle de acesso baseado em risco

**🛡️ Identity Protection** <br>
Detecção de login suspeito<br>
Mitigação de ataques de credential stuffing e brute force<br>
Proteção baseada em risco (risk-based authentication)

**☁️ Zero Trust Security Model** <br>
Princípio de “never trust, always verify”<br>
Acesso baseado em contexto e identidade<br>
Segmentação de acesso a recursos críticos

**🛠️ Ferramentas e Tecnologias** <br>
Microsoft Entra ID<br>
Microsoft Defender for Identity (quando aplicável)<br>
Microsoft Sentinel (correlação de logs de login)<br>
Azure Active Directory logs<br>
KQL (Kusto Query Language)<br>
Conditional Access Policies<br>
MFA / Security Defaults
