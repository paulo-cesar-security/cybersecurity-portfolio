# 🎥 Análise Forense de Vídeo Digital

**📌 Visão Geral**

Este projeto apresenta um procedimento completo de análise forense de vídeo, com foco na verificação de autenticidade e identificação de possíveis manipulações em evidências digitais.

O cenário simula um contexto jurídico real, onde um arquivo de vídeo é submetido à análise técnica para validação pericial.

**🎯 Cenário**

Um advogado encaminha, via e-mail, um arquivo de vídeo que supostamente registra um fato relevante para um processo judicial.

A partir disso, foi conduzida uma análise forense com o objetivo de verificar a integridade, autenticidade e possíveis indícios de adulteração do material.

**🔍 Procedimento Realizado**

Durante a investigação, foram executadas as seguintes etapas:

🔐 Preservação da evidência digital original<br>
💾 Criação de cópia de trabalho (cópia forense)<br>
🔑 Cálculo de hash criptográfico para garantia de integridade<br>
📊 Análise de metadados do arquivo de vídeo<br>

🎞️ Identificação de:<br>
Codec utilizado<br>
Taxa de quadros (FPS)<br>
Resolução<br>
Estrutura de compressão<br>

✂️ Análise de possíveis:<br>
Cortes<br>
Edições

Recompressões<br>
🧩 Inspeção de frames individuais para detecção de inconsistências<br>
⏱️ Verificação de continuidade temporal e estrutural<br>
🧬 Identificação de artefatos de edição ou sobreposição digital<br>
🧾 Documentação completa de todo o processo<br>
📄 Resultado

Ao final da análise:

Foi elaborado um Laudo Técnico Pericial<br>
O laudo indica a existência (ou não) de:<br>
Manipulação<br>
Edição<br>
Adulteração do vídeo<br>
As conclusões foram fundamentadas em evidências técnicas documentadas<br>
