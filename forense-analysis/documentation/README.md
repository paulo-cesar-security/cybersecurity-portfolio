📚 Documentation — Perícia forense Computacional

Bem-vindo ao meu repositório de documentação de Forense. Este espaço foi criado para centralizar anotações, procedimentos, pesquisas, comandos, estudos e referências utilizadas no dia a dia da área de um perito Forense computacional.

O objetivo é manter uma base organizada, rápida de consultar e continuamente atualizada para apoio em atividades técnicas, laboratoriais e profissionais.
